# **App Name**: Florarte Online

## Core Features:

- Product Details: Display products with multiple images, name, price, description, and dimensions.
- Date Selection: Calendar integration to select shipping date with restricted past dates.
- Time Slot Selection: Display available delivery time slots and validate selection before adding to cart.
- AI Recommendations: Display of similar products using the AI 'tool' for context, to assist the user in finding alternative options.
- Contact Section: Display of contact information, including a form, map, phone number, email, and a floating WhatsApp button.
- Shopping Cart: Implementation of a shopping cart with detailed visualization and coupon application.

## Style Guidelines:

- Primary color: Rose color (#F472B6) to convey warmth, love, and appreciation, aligning with floral gifts.
- Background color: Light pink (#FCE4EC), a very desaturated version of the primary, providing a soft, clean backdrop.
- Accent color: A slightly purple color (#D81B60) that enhances CTAs, analogous to the rose.
- Body and headline font: 'Alegreya', a serif font to convey the vintage personality and give the app a fashionable feel.
- Use elegant, minimalist icons for categories and user actions.
- Mobile-first responsive design for optimal viewing on all devices.
- Subtle animations and microinteractions to enhance user engagement.
